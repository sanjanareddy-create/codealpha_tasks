# Basic Chatbot

## Description
A simple rule-based chatbot built using Python. It responds to predefined user inputs.

## Features
- Responds to "hello"
- Responds to "how are you"
- Responds to "bye"
- Uses loops and conditional statements
- Easy to understand

## Technologies Used
- Python

## How to Run

1. Open terminal.
2. Navigate to project folder.
3. Run:

```bash
python chatbot.py
```

## Example

Input:

hello

Output:

Hi!

Input:

how are you

Output:

I'm fine, thanks!

Input:

bye

Output:

Goodbye!
