# Hangman Game

A simple text-based Hangman Game developed using Python.

## Features

* Random word selection
* Letter-by-letter guessing
* 6 incorrect attempts allowed
* Win and lose conditions
* Console-based gameplay

## Technologies Used

* Python 3

## How to Run

```bash
python hangman.py
```
