# Stock Portfolio Tracker

A simple Python application that calculates the total value of a stock portfolio using predefined stock prices and user-entered quantities.

## Features

* Stores stock prices using a Python dictionary
* Accepts stock names and quantities from the user
* Calculates total investment value
* Displays investment value for each stock
* Saves the portfolio summary to a text file

## Technologies Used

* Python 3

## How to Run

1. Open a terminal in the project folder.
2. Run the program:

python stock_tracker.py
